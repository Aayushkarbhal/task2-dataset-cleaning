Data Cleaning & Missing Value Handling

Dataset Used
House Prices Dataset

Objective
To clean the dataset by identifying and handling missing values in order to improve data quality and make it suitable for machine learning models

Steps:
Loaded the dataset using Pandas
Identified missing values using isnull()
Handled missing numerical values using median imputation
Handled missing categorical values using mode imputation
Removed columns with very high missing values
Validated the dataset after cleaning
Saved the cleaned dataset for further use
